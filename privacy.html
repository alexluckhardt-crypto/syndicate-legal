Privacy Policy

Last updated: March 29, 2026

Syndicate ("the App", "we", "us") is a desktop application that helps creators schedule and publish video content to social media platforms. This policy explains what data we handle and how.

1. Data We Collect

Syndicate does not collect, transmit, or store any personal data on external servers. All data — including API keys, access tokens, and scheduled post information — is stored locally on your device in:

  • A local .env file (credentials)
  • A local SQLite database (your post queue)

2. Third-Party Services

When you connect your accounts and post content, the App communicates directly with the following third-party platforms on your behalf:

  • Meta (Facebook & Instagram) — via the Meta Graph API
  • YouTube — via the YouTube Data API v3
  • Pinterest — via the Pinterest API v5
  • LTK (LikeToKnowIt) — via the LTK creator portal
  • ManyChat — via the ManyChat API
  • Anthropic — for AI caption generation via the Claude API

Each of these services has its own privacy policy. We encourage you to review them. We only send data to these services as needed to carry out actions you explicitly request.

3. OAuth Tokens

When you authenticate with Facebook, YouTube, or Pinterest, the App receives an access token and/or refresh token. These tokens are stored only on your local device and are never transmitted to our servers.

4. Uploaded Content

Videos and captions you publish through the App are sent directly to the relevant social media platforms. We do not retain copies of your content.

5. Children's Privacy

Syndicate is not directed at children under 13. We do not knowingly collect data from children.

6. Changes to This Policy

We may update this policy from time to time. The "Last updated" date at the top will reflect any changes. Continued use of the App after changes constitutes acceptance.

7. Contact

If you have questions about this policy, please open an issue or reach out via GitHub.
